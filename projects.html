NetSentinal ‚Äì AI-Powered Secure Network

NetSentinal is an offline, AI-powered secure network system designed for jammer-affected high-security zones such as prisons, defense bases, war zones, borders, and research labs. When jammers block external communication, essential services fail. NetSentinal solves this by creating its own secure local Wi-Fi network that works without internet.

The system continuously captures and analyzes network traffic of all connected devices. An AI model detects anomalies, VPN tunneling, suspicious domains, and unauthorized behavior in real-time. The admin dashboard displays all devices, logs, alerts, and allows one-click actions like block, unblock, and disconnect.

With automated threat response and a fully offline architecture, NetSentinal ensures complete security and uninterrupted operation even under heavy jammer interference.
